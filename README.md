# 宅音乐播放器
--------
宅音乐播放器，HTML5网页播放器，集成后台管理及API调用，目前正在开发中，敬请关注
## 技术栈
- 后端：thinkphp 5.1
- 前端：layui
- 数据库：mysql

## 安装
### 依赖
- composer
- php 5.6+
- mysql 5.5+
### 步骤
安装php依赖包
```
composer install
```
配置数据库，配置链接数据库名以及用户名密码
````
/config/database.php
````
